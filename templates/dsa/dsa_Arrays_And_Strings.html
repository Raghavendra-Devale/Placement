Arrays and Strings Problems

Practice these fundamental array and string manipulation problems with detailed explanations

Problem Description Difficulty Practice Link Video Solution Status
Two Sum Given an array of integers and a target sum, find two numbers that add up to the target. For example, in array [2,7,11,15] with target 9, return [0,1] since 2+7=9. This teaches you how to efficiently search through arrays. Easy LeetCode Watch Solution
What is an Anagram? An anagram is when two words have the exact same letters but in different orders. For example, "listen" and "silent" are anagrams. "heart" and "earth" are anagrams. This problem teaches you to compare the character composition of strings. Easy LeetCode Watch Solution
Reverse Array Write a program to reverse an array. For example, [1,2,3,4] becomes [4,3,2,1]. This fundamental problem teaches you how to manipulate array elements and use the two-pointer technique (working from both ends simultaneously). Easy GeeksforGeeks Watch Solution
Palindrome Check Check if a string reads the same forwards and backwards. For example, "racecar" is a palindrome. This teaches basic string comparison and handling special characters/spaces. Easy LeetCode Watch Solution
Find Unique Characters Find the longest substring without repeating characters. For example, in "abcabcbb", the longest substring is "abc" (length 3). This teaches you about sliding window technique - a way to process strings piece by piece efficiently. Medium LeetCode Watch Solution
Rotate Array Rotate an array to the right by k steps. For example, [1,2,3,4,5] rotated by 2 becomes [4,5,1,2,3]. This teaches array manipulation and understanding circular movement in arrays. Medium LeetCode Watch Solution
Water Container Given heights of walls [1,8,6,2,5,4,8,3,7], find the two walls that can hold the most water between them. This teaches you how to solve geometric problems using arrays and optimize using two pointers. Medium LeetCode Watch Solution
Group Similar Words Given ["eat","tea","tan","ate","nat","bat"], group related words together: [["eat","tea","ate"],["tan","nat"],["bat"]]. This teaches you how to organize strings based on their character composition using hash maps. Medium LeetCode Watch Solution
Three Numbers Sum Find three different numbers in an array that add up to zero. For example, in [-1,0,1,2,-1,-4], one solution is [-1,-1,2]. This builds on Two Sum and teaches advanced array manipulation. Medium LeetCode Watch Solution
String Compression Compress a string by replacing repeated characters with their count. For example, "aabbbcccc" becomes "a2b3c4". This teaches string building and counting consecutive elements. Medium LeetCode Watch Solution
Smallest Window Search Find the smallest part of a string that contains all characters from another string. For example, in string "ADOBECODEBANC", find smallest window containing all letters of "ABC". This teaches advanced string processing using sliding window. Hard LeetCode Watch Solution
Longest Palindromic Substring Find the longest substring that reads the same forwards and backwards. For example, in "babad", the answer is "bab" or "aba". This teaches dynamic programming and string manipulation techniques. Hard LeetCode Watch Solution
